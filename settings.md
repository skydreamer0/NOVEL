# 小說企劃設定集 (Novel Project Bible)

## 0. 基本資訊 (Project Info)
*   **書名 (暫定)**：《深淵駭客：凡人代碼》 / 《我將修仙界格式化》
*   **類型**：#奇幻修仙 #克蘇魯/深淵 #賽博龐克系統 #反烏托邦
*   **字數規劃**：長篇連載 (1000章+)
*   **核心標籤**：理性升級、數據流、詭異懸疑、智商碾壓。

## 1. 核心概念 (Core Concept)
*   **一句話梗概 (Logline)**：一名凡人駭客穿越到一個被深淵侵蝕、歷史被篡改的修仙世界，在眾人皆以「向下飛升」墮入深淵為榮的瘋狂中，他必須藉助能轉化深淵之力的賽博系統，駭入這個世界的源代碼，扭轉錯誤的真理。
*   **核心衝突**：
    *   **表層**：主角唯一清醒 vs 全世界集體瘋狂。
    *   **深層**：冰冷的機械邏輯 (System/Order) vs 扭曲的血肉深淵 (Abyss/Chaos)。

## 2. 世界觀設定 (World Building) - "扭曲的真理"

### 2.1 背景概述
*   深淵入侵後，修仙界歷史被改寫，**「變異」=「進化」，「瘋狂」=「頓悟」**。
*   天空是紫紅色的，月亮散發詭異的紫色光芒。
*   植被大多是黑色的，經常發出詭異的聲響。

### 2.2 關鍵設定 - "飛升" (Ascension)
*   **世人認知**：飛升 = 向下深入地底。越接近地底深淵，靈氣越濃（其實是瘴氣）。
*   **真相**：這是獻祭給古老邪神「深淵母神」的過程。

### 2.3 修煉真相
*   **本質**：修煉 = **主動異變** + **精神崩壞**。
*   **限制器 (The Limiter)**：植入體內的「鎮魂釘」，讓人產生幻覺，以為自己修的是仙氣，其實身上長滿了毛。
*   **等級對照**：
    1.  **寄生期 (Parasitic Stage)** / 練氣：深淵孢子寄生。
    2.  **異結期 (Node Formulation)** / 築基：體內長出異體器官。
    3.  **裂變期 (Fission)** / 金丹：肉體不穩定分裂。
    4.  **不可名狀期 (The Unspeakable)** / 元嬰：脫離人形。

## 3. 長生宗設定 (Changsheng Sect)

### 3.1 宗門概述
*   長生宗的真正目的並非修煉飛升，而是**育種**。
*   整個宗門就是一個巨大的培養皿，所有弟子都是用來培育「深淵種子」的養料。
*   那些所謂的「飛升」，其實是被獻祭給深淵母神的儀式。
*   高層修士們之所以能夠長生，是因為他們已經和深淵母神簽訂了契約，成為了「祂」在這個世界的代言人。

### 3.2 宗門結構
```
掌門 (元嬰期，從未露面)
    ├── 內門
    │   ├── 五名長老 (築基期以上)
    │   │   ├── 大長老
    │   │   ├── 二長老
    │   │   ├── 三長老 (周長老) ← 與外門有秘密交易
    │   │   ├── 四長老
    │   │   └── 五長老
    │   └── 內門弟子
    └── 外門
        ├── 四名執事 (築基期)
        │   ├── 趙執事 (已死) - 築基中期，管理東區
        │   ├── 錢執事 - 築基後期，擅長追蹤，與趙執事有血緣關係
        │   ├── 孫執事 - 膀大腰圓的胖子，精明
        │   └── 李執事 - 最年輕但最強
        └── 外門弟子
            ├── 普通弟子
            └── 雜役弟子 (最底層)
```

### 3.3 外門地點
*   **外門弟子居住區**：雜役弟子居住的破舊區域。
*   **深淵區**：外門最危險的區域，專門清理深淵污染物，死亡率極高。
*   **後山**：變異獸出沒區域，是外門弟子心目中的禁地，也是主角的狩獵場。
*   **執事殿**：外門四執事辦公場所。

## 5. 金手指：賽博龐克系統 (The Cyberpunk System)

### 5.1 系統代號
**【內核 (The Kernel)】**

### 5.2 視覺介面 (HUD)
*   **風格**：極致簡約的**藍/綠色霓虹線條**，懸浮在充滿血腥與爛肉的視野中，形成強烈對比。
*   **系統訊息格式**：使用【】框起訊息。
*   **動態標記**：深淵怪物頭上會顯示威脅等級、弱點標記等。

### 5.3 核心功能 (Modules)

#### 1. 防火牆 (Firewall / SAN Guard)
*   **功能**：自動過濾「不可名狀」的畫面，將其馬賽克化或數據化，防止主角SAN值歸零。
*   **機制**：消耗能量維持。若能量耗盡，防火牆破碎，主角將直視深淵（危機）。

#### 2. 解析之眼 (Analytical Eye)
*   **功能**：掃描分析目標，顯示威脅等級、弱點、技能等資訊。
*   **當前版本**：V1.5，可分析築基期目標。

#### 3. 生物反應爐 (Bio-Reactor)
*   **功能**：吸收被污染的深淵物質/屍體，轉化為純淨的通用能量點數。
*   **特殊用法**：可用嘴直接「吃」敵人來獲取能量。
*   **過濾功能**：三重過濾可淨化深淵污染。

#### 4. 駭入協議 (Hacking Protocol)
*   **功能**：針對敵人的「限制器 (Limiter)」或陣法進行攻擊。
*   **已展示用法**：
    *   注入垃圾數據使敵人短暫混亂
    *   入侵陣法控制核心，獲取權限
    *   在物品上偽造能量痕跡

#### 5. 全息投影 (Holographic Projection)
*   **功能**：改變自身外觀，偽裝成虛弱/正常狀態。
*   **消耗**：1點能量/小時

#### 6. 編譯器 (Spell Compiler)
*   **功能**：將修仙功法/法術解析為「代碼腳本」，可進行改造過濾。
*   **已改造功法**：
    *   吸血箭術 (改) - B級，吸血+攻擊循環
    *   黑霧遁 (改) - 隱匿功法，已過濾深淵污染
    *   腐骨掌 (改) - 腐蝕功法

### 5.4 系統限制：負載與過熱 (System Load & Overheat)
*   **核心邏輯**：主角的凡人肉體（硬體）無法長時間承受系統的高強度運算（軟體）。
*   **負載值 (Load)**：
    *   使用任何功能（解析、轉化、駭入）都會累積「系統負載」。
    *   **安全區 (0-60%)**：正常運作。
    *   **過熱區 (60-90%)**：頭痛欲裂，鼻孔流血，與系統連接不穩定。
    *   **崩潰區 (90-100%)**：大腦燒毀，直接死亡或變成植物人。
*   **冷卻機制**：需要通過**睡眠**或**特定藥劑（冷卻液）**來降低負載。
